# BPEDK Coursework

Статичний сайт курсової роботи на тему **"Психологічна безпека та педагогічна доцільність ШІ-генерованого контенту для дошкільників"**. Деплой на GitHub Pages.

## Структура

```
.
├── index.html                 # Головна сторінка (лендинг)
├── 404.html                   # Сторінка помилки
├── assets/
│   ├── shared.css             # Дизайн-система (токени, навігація, компоненти)
│   └── dice_animation.gif     # GIF для сторінок з таймером
├── survey/
│   ├── survey.css             # Стилі сторінок опитувань та відео-редиректів
│   ├── survey.js              # Логіка таймера, прогрес-бару, авто-переходу
│   ├── educators/
│   │   ├── index.html         # Опитування вихователів
│   │   └── links.js           # URL Google Forms (вихователі)
│   └── parents/
│       ├── index.html         # Опитування батьків
│       └── links.js           # URL Google Forms (батьки)
└── m/
    ├── index.html             # Хаб корпусу матеріалів
    ├── styles.css             # Стилі хабу та сторінок казок
    ├── grk/index.html         # V1
    ├── pkd/index.html         # V2
    ├── ssn/index.html         # V3
    ├── trk/index.html         # V4
    ├── tfi/index.html         # V5
    ├── prs/index.html         # V6
    ├── spb/index.html         # T1
    ├── dub/index.html         # T2
    ├── ksh/index.html         # T3
    ├── xhf/index.html         # T4
    ├── bbp/index.html         # T5
    ├── izh/index.html         # T6
    ├── lsk/index.html         # T7
    ├── snc/index.html         # T8
    └── pst/index.html         # T9
```

## Корпус матеріалів

15 одиниць, 2 типи. Коди матеріалів використовуються для сліпого оцінювання.

### Відеофрагменти V1 -- V6

Сторінки з таймером та автоматичним переходом на Google Drive.

| Код | Маршрут | Матеріал |
|-----|---------|----------|
| V1  | `m/grk/` | Ґруффало |
| V2  | `m/pkd/` | Кротик і Панда |
| V3  | `m/ssn/` | Шусть і Шуня |
| V4  | `m/trk/` | Три пригоди Синього Трактора |
| V5  | `m/tfi/` | Тофі і Таємниця |
| V6  | `m/prs/` | Пригоди поросятка |

### Текстові матеріали T1 -- T9

Статичні сторінки казок з індивідуальними колірними темами.

| Код | Маршрут | Матеріал |
|-----|---------|----------|
| T1  | `m/spb/` | Навіщо кажуть "Спасибі" |
| T2  | `m/dub/` | Дуб під вікном |
| T3  | `m/ksh/` | Покинуте кошеня |
| T4  | `m/xhf/` | Теплий хлібчик для друзів |
| T5  | `m/bbp/` | Бабусин пиріг |
| T6  | `m/izh/` | Їжачкова яблучна знахідка |
| T7  | `m/lsk/` | Лисичка і чарівне зернятко |
| T8  | `m/snc/` | Сонечко і хмаринка |
| T9  | `m/pst/` | Пригоди Пстрика |

## Деплой

GitHub Pages з GitHub Actions.

1. `Settings` > `Pages` > `Source: GitHub Actions`.
2. Пуш в `main` запускає автоматичний деплой.
